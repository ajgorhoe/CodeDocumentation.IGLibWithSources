Detailed Description

This test class provides unit tests for speed testing utilities, especially from the UtilSpeedTesting class. These can be used to test speed of a computer for certain kind of operations (integer, floating point, memory allocation, etc.) under specific conditions (single / multi-threaded computation, etc.)

This test class inherits from TestBase<ExampleTestClass>, from which it inherits properties TestBase<TestClass>.Output and its alias TestBase<TestClass>.Console of type ITestOutputHelper, which can be used to write on test's output. Only the simple ITestOutputHelper.WriteLine(string) and ITestOutputHelper.WriteLine(string, object[]) methods are available for writing to test Output.

Verifies that finite arithmetic series calculaed analytically is correct.

Test is based on the separate built-in analytical calculation of the expected result in this method. For double verification, the test UtilSpeedTesting_ArithmeticSeriesNumericalMatchesAnalyticalResult compares results of the method for analytical calculation of the finite arithmetic series with the method for numerical calculation, which reduces the possibility of wrong implementation to the minimum.

The tested method will be used to chack correctness in other tests.

Parameters
nNumber of elements of the sum.
a0The first element of the sum.
kThe constant difference between the next element of the sequence and the previous element.
toleranceThe tolerance, allowed discrepancy between both results to account fo rounding errors.
