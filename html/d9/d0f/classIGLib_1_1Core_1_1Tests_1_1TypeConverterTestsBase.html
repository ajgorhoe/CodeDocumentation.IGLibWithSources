Detailed Description

Base class for type converter testing classes such as BasicTypeConverterTests.

Provides generic implementation of typical test methods for testing round-trip or one direction conversions, and helper stuff like example classes on which conversions can be tested, with prescribed relations (inherits from, implements implicit / explicit conversion, etc.).

Performs a speed test of conversion via ITypeConverter from a value of type OriginalType to an object variable of target type TargetType and back to value of type RestoredType .

Parameters
typeConverterConverter used for type conversion.
originalValueOriginal value that is converted converted to an object of type TargetType.
expectedAssignedObjectValueThe expected value after conversion of the originalValue to a value of type TargetType and assignment to a variable of type object
expectedRestoredValueExpected restored value after conversion of the original value and storing it in a variable of type object and restoring it from the object variable to a variable of type RestoredType .
restoreObjectBackToValueIf true (which is default) then object is also restored back to a value of type RestoredType .
Template Parameters
OriginalTypeType of the origial value to be stored as object.
TargetTypeType to which the originalValue will be converted when being stored in an object variable.
RestoredTypeType of variable to which the value will be restored from the variable of type object.

Expect for parameters related to measurement of the speed of conversions, parameters have the same meaning as in the TypeConverter_ConversionToObjectAndBackTest<OriginalType, TargetType, RestoredType>(ITypeConverter, OriginalType, TargetType, RestoredType, bool) method. Type parameters also have the same meaning as in that method.

Parameters
numExecutionsNumber of executinos for speed measurements.
