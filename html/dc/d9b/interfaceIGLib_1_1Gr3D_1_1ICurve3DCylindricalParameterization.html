Detailed Description

Cylindrical parameterization of a curve in 3D. Beside the CurveCylindrical and (optional) CurveDerivativeCylindrical, it also contains bounds on parameter, StartParameterCylindrical and EndParameterCylindrical as part of the class.

double IGLib.Gr3D.ICurve3DCylindricalParameterization.rhoDerivative

3D vector function of scalar argument, which represents the derivative of CurveCylindrical with respect to parameter. In each point on curve (for each parameter value), this defines a tangent on the curve (or speed vector), with its size representing the velocity.
