override Type BindToType (string assemblyName, string typeName)
 Specifies the mapping of assemblyName and typeName of the serialized object into a type of an object into which it deserializes. This mapping is defined by a collection of elementary ISerializationBinder objects derived from SerializationBinderBase class, each of which defines its own mapping. The overall mapping is defined by this method in such a way that it iterates (in order of elements) over all serialization binders on the ContainedBindersPrivate, and returns as target type the first non-null Type object returned by the ISerializationBinder.BindToType(string, string) method returned by any of the objects contained in ContainedBindersPrivate called with the same parameters assemblyName and typeName . Detailed Description

A cascaded SerializationBinder class that maps the assembly and type information of the serialized objects into classes into which objects shoulld be deserialized, based on definitions from a collection of the contained elementary ISerializationBinder classes. Similar to SerializationBinderCascadedImmutable, except that binders can also be added (at the end of the list) after creation.
