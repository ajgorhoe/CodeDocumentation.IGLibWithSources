Detailed Description

Registry of types with a common base type, similar to DerivedTypesRegistry<BaseType, TypeInfoBase>, with the simple type info class TypeInfoBase and with added registration method based on type (RegisterDerivedType<TypeToRegister>). Each type in is registered with a type info object (of type TypeInfoBase and can be registered under differeent names (typically, simple type names and qualified names).

Template Parameters
BaseTypeBase type for types that can be registered in the current registry. Registered types must derive from this type.
