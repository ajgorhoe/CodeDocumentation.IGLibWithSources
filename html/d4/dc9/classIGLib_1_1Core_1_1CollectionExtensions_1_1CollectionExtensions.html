Detailed Description

Various extension methods for collections. In particular, conversion for readable strings, which produce readable outputs in application or assist in testing.

ToReadableString (ToReadableString(object, string, string, string, string) and similar):

\

Conversions of various array-like objects to readable strings. Various types of collections have their own overloads, which are statically dispatched (e.g. IList<T>, IEnumerable<T>, T[], rectangular arrays T[,], T[,,], jagged arrays T[][], T[][][]).

There is also the method that detects the type of array-like objects dynamically, ToReadableString(object, string, string, string, string), and produces, according to the actual type of the argument, a similar output for array-like type of objects than the corresponding statically typed counterparts.

The ToreadableString(...) type of methods are meant to be used as static extension methods, but certain formatting aspects are parameterized by static properties ArrayBracketOpen, ArrayBracketClosed, ArraySeparator and ArrayIndentation. Each of these methods has a set of optional parameters where these formatting parameters can be customized at will, and the mentioned properties represent default values for these parameters. This increases reusibility, and if necessary, the equivalent instance classes whose instances are injectable could easily be created from the current class.

Converts the specified object o to a readable string.

This method dynamically checks the actual type of o . If the object is an array, 2 or 3 dimensional rectangular array, 2 or 3 dimensional ljagged array, a list (it implements the IList interface) or enumerable (it implements the IEnumerable interface, then the appropriate method is called to produce the string representation suitable for array-like objects.

Parameters
oObject to be converted to a readable string.
Returns
Ths string containing a representation of the o , indentation String used for single indentation; default is ArrayIndentation openBracket String used as open bracket in array representation; default is ArrayBracketOpen closedBracket String used as closed bracket in array representation; default is ArrayBracketClosed separator String used as separator in array representation; default is ArraySeparator which is also nicely structured and readable for objects of array-like types.
