Detailed Description

A simple SerializationBinder, defines a single mapping from the original assembly and type name to a target type. Matching of assembly and type name is literal and case sensitive.

Use SerializationBinderCascaded to combine mulltiple binders of this type in order to extend the mapping.

Defines a single mapping from the original assembly and type name to the target type. Matching of assembly and type name is verbatim and case sensitive.

Parameters
assemblyNameAssembly name as found in serialized object, must match the OriginalAssemblyName property verbatim in order to take effect (meaning that the property must provide a full assembly name).
typeNameType name as found in serialized object, must match the OriginalTypeName property verbatim to take effect (meaning that the property must provide a namespaced type name).
Returns
null when assemblyName and typeName arguments don't match the corresponding interna properties (OriginalAssemblyName and OriginalTypeName), or the mapped type (value of property TargetType) when matching occurs.
