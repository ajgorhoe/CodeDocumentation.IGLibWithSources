Detailed Description

A SerializationBinder, defines a single mapping from the original assembly and type name to a target type. Matching of assembly and type name may be literal (as in SerializationBinderSimple) but can be fine tunned with parameters.

Type is immutable.

Properties used to fine tune the behaior:

IsAssemblyNameIgnored

IsAssemblyNameIgnored

IsTypeNameSimple

Use SerializationBinderCascaded to combine mulltiple binders of this type in order to extend the mapping.

Defines a single mapping from the original assembly and type name to the target type. Matching of assembly and type name is verbatim and case sensitive.

Parameters
assemblyNameAssembly name as found in serialized object, must match the SerializationBinderSimple.OriginalAssemblyName property verbatim in order to take effect (meaning that the property must provide a full assembly name).
typeNameType name as found in serialized object, must match the SerializationBinderSimple.OriginalTypeName property verbatim to take effect (meaning that the property must provide a namespaced type name).
Returns
null when assemblyName and typeName arguments don't match the corresponding interna properties (OriginalAssemblyName and OriginalTypeName), or the mapped type (value of property TargetType) when matching occurs.
