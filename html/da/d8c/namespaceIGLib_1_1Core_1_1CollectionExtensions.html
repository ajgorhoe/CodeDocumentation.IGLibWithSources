static string IGLib.Core.CollectionExtensions.ToReadableString ( this char  ch)
static

Converts a character to a readable string.

The returned string contains the character within single quotes. If the character itself is a single quote, then it is escaped by a backslash (this is not true for any other characters).

Parameters
chCharacter to be converted.
