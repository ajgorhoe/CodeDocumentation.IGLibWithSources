Detailed Description

Similar as ModelParameterSetFixedBase<ModelParameterType>, except that parameter objects can be added, replaced or removed after the parameter set is created.

Template Parameters
ModelParameterTypeType of parameter objects contained in the set, must implement the IModelParameter interface. Commonly, the ModelParameter and the generic ModelParameter<ParameterType> are used.
Type Constraints
ModelParameterType :IModelParameter
