Generates a tubular mesh using the Parallel Transport Frame (PTF) and numerical tangent vectors.

Parameters
curveDefinition of the parametric curve used to generate the tubular surface.
tStartStarting parameter.
tEndEnding parameter.
radiusRadius of the circular tube.
numCurvePointsNumber of mesh points along the curve.
numCirclePointsNumber of mesh points around the circumference of the tube.
hrelSpecification of step for numerical differentiation, relative to the meshing step (difference in parameter between meshing points).
epsSpecifies how close to parallel to a tangent can a unit vector be at most, to still be eligible to generate the first normal to the tangent via orthogonalization. Should be less than 0.5; normally it should be less than 0.2, also 0.05 could be a good value. If scalar product of the tangential unit vector and the candidate unit vector minus 1.0 is less than eps then the candidate vector is not taken into account because it is too parallel to the tangent.
normalizeFromPreviousIf true then the second normal is calculated from previous normal by orthogonalization. Otherwise, it is calculated by a vector product of the tangent and the first normal. Default is false.
restrictToIntervalWhether evaluations of the curve points must be limited to the meshing interval. If true then the step for numerical differentiation must be taken in such a way that all vurve evaluations fall within the meshing interval, end ponts inclusive. Ths is useful when the function has a singularity or sidcontinuity just outside the meshing interval.

Generates a tubular mesh from the specified parametric curve using the Frenet frame and numerically calculated tangent vector.

This method of generating the mesh will lead to a "pinched pipe mesh" in ponts where the swcond derivative changes direction because the normals that define meshning around circumberence change direections by full angle (180 degrees).

For parameter descriprions, see GenerateMesh(Func<double, vec3>, double, double, double, int, int, double, double, bool, bool).

Generates a tubular mesh using the Frenet frame and analytical or fallback tangent function.

The same as GenerateMeshByFrenet(Func<double, vec3>, Func<double, vec3>, double, double, double, int, int, double), except that analytical curve derivative (tangent) is used (specified by tangent ) and we don't need to calculate derivatives numerically.

For parameter descriprions, see GenerateMesh(Func<double, vec3>, double, double, double, int, int, double, double, bool, bool).
