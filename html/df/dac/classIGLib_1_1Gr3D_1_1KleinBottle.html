◆ ff

double IGLib.Gr3D.KleinBottle.ff
get

The number of half-twists by which the generating figure-eight curve is rotated by 180 degrees (π/2). before moving along the full circle (angle 2*π). Default is ffDefault.

Only for odd integer values of ff the represented surface is non-orientable Klein bottle (for odd values, we glue one surface of the curved stripes of annulus with the other, such that traveling on any side by two full angles reaches all segments of the surface, which has a single side). For ff = 0 we get an figure-eight torus. For even integer values (2, 4, 6, ...) the twisted surface remains orientable.

This parameter is intentionally not an integer, such that we can use it as continuous parameter in evolution of the surface from figure-eight torus (ff = 0) to 1 half-twist, 3 half-twists, etc., Kein bottles.
