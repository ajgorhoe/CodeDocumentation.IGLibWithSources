Tries to parse an object of the specified type T and store it to the provided reference.

This method uses reflection to check whether the type T contains the method TryParse. If it does, it executes that method via reflection, writes the result to result , and returns true. If it does not, it just returns false.

Template Parameters
TType of value / object to be parsed from inputString .
Parameters
inputStringString that should hold the representation of the object that can be parsed.
resultReference of the variable where the parsed object is stored when parsing is successful.
Returns
True if parsing was successful, false otherwise.
