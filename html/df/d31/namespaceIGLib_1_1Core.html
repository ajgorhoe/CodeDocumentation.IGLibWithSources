delegate ref TResult IGLib.Core.FuncRef< TResult > ( )

A generic delegate that returns a reference to an object of the specific type.

Similar to Func<TResult>, but instead of a value (object) of type T it returns a reference to a variable or field of this type.

Returned value can be used to set a variable of type TResult or of type "ref <typeparamref name="TResult"/>", or as return of a method with such reference return type.

Template Parameters
TResultType of object whose reference is returned by the delegate.
Returns
A reference to an object of type T, through which the object can be accessed or set.
