Detailed Description

Abstract base implementation of ISingleTypeConverter<TSource, TTarget>. Sets SourceType and TargetType properties according to generic type parameters TSource and TTarget , respectively, and implements all methods by calling their counterparts. Implementatkion based on this base class must override and properly implement either the TryConvertTyped(TSource, out TTarget) or the ConvertTyped(TSource) method.

Template Parameters
TSourceType of the objects that can be converted to TTarget by the current converter.
TTargetType to which source object of type TSource are converted by the current converter.
