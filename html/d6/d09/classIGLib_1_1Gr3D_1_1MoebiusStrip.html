Detailed Description

Parametric Möbius strip, with configurable number of half-twists ff (which must be odd to generate non-orientable surface).

Generated as ruled surface where a line segment of length 1 moves along the circle with radius aa and rotates in such a way that after the whole circle is visited, the final rotation is by ff half angles (180 degrees).

Parameters: radius aa, number of half-twists ff (the strip surface is non-orientable for odd values 1, 3, 5, ...; 1 for the standard strip with one half-twist)

◆ ff

double IGLib.Gr3D.MoebiusStrip.ff
get

The number of half-twists by which the generating line is rotated by 180 degrees (π/2). Default is ffDefault.

Only for odd integer values of ff the represented surface is non-orientable (for odd values, we glue one surface of the annulus with the other, such that traveling on any side by two full angles reaches all segments of the surface, which has a single side). For ff = 0 we get an annulus

. For even integer values (2, 4, 6, ...) the twisted surface remains orientable.

This parameter is intentionally not an integer, such that we can use it as continuous parameter in evolution of the strip from annulus (ff = 0) to 1 half-twist, 3 half-twists, etc.
