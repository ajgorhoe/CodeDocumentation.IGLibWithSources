Detailed Description

Converts values from a single specific source type (SourceType ) to a single specific target type (TargetType ). Conversion methods are not typed and have parameters and return value of type object.

Converts values from a single specific source type (SourceType) to a single specific target type (TargetType). Conversion methods are not typed and have parameters and return value of type object.

Use the generic ISingleTypeConverter<SourceType, TargetType> counterpart in order to include mathods with typed parameters.
