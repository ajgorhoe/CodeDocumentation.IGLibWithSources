Detailed Description

This class is OBSOLETE and is replaced by chasses that implement ITypeConverter. The class will be kept for some time for reference. You can remove it at least in Nov '25, possibly before (if you know it is not useful any more).

Handles type conversionss where the source type is inferred from the source object and the target type is specifyed by the Type object rather than generic type parameter.
