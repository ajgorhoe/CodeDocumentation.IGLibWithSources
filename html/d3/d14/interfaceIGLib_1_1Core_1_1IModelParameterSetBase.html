Detailed Description

A set of model parameters of the specified type, where parameters can be added, removed or changed after initialization of the set.

This can represents all parameters that are used in the certain parameterized model, or a related group of parameters.

Template Parameters
ModelParameterTypeSpecifies the type of parameter objects contained in the parameter set (need to implement the IModelParameter interface).
