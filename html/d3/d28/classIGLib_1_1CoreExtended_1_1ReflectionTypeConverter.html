Converts a value to the specified target type, supporting collections. Conversion is bsed on the actual type of value and the specified target type (targetType ). See the class description for more details.

Takes a value of type object, identifies its actual type, converts it to the type compatible with the specified desired target type, and returns the converted value as object.
