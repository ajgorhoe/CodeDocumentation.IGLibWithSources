Detailed Description

Provides read/write access to elements of a fixed array. This is typically used to access array-like internal elements of other classes, supported by an internal underlying field or variable of type ElementType[].

IMPORTANT: To use this class, the underlying array may not be reallocated / resized during the lifetime of this class's object. If the underlying array may change, use the ArrayAccessorRef<ElementType> instead.

Template Parameters
ElementTypeType of elements of array to which access is provided by objects of this class.
