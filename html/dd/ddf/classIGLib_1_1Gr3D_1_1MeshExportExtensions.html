Exports the mesh to GLF format.

Features:

* Embedded base64 buffer (ASCII .gltf)

* Vertex positions

* Triangle indices

* Normals (optional, from VertexNormals)

* Vertex colors (optional, from VertexColors)

* Material reference

* Optional lights (LightSource)

*

*

Parameters
meshMesh to be exported.
gltfPathName of the export file.
materialMaterial to be assigned to the surface.
lightsLights used in rendering the surface, will be written to the file if specified.
